# Apex inst. guide:
1. Compile main.py:
  py -m pip install pyinstaller
  pyinstaller --onefile --console --name apix main.py
2. Move your apix.exe to PATH or System32. 
3. Change your directory, and open file by writing this command
  apix <FILE_name>.a 
# How to open file without compilation: 
1. You can also dont compile main.py file, and move your exe file into the PATH / System32
